Portfolio Website

A modern developer portfolio built with Next.js, TypeScript, and Tailwind CSS. It features a clean UI, MDX‑powered blog, dynamic project pages, optimized images, and a responsive layout. Designed for performance, readability, and easy content updates.

Tech Stack
- Next.js (App Router)  
- React  
- TypeScript  
- Tailwind CSS  
- MDX  

Development
```
npm install
npm run dev
```

Build
```
npm run build
```

Deployment
Optimized for Vercel. Upload the project root (excluding `.next/` and `node_modules/`) for deployment.
